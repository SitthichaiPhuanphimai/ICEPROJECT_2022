# ICEPROJECT_2022
ICE PROJECT 2022 DAT.1


2 player shooter

The goal is to shoot the other player


Movement for Player 1

Move up - W
Move down - S
Move left - A
Move right - D
Shoot - SPACE

Movement for Player 2

Move up - Key Up
Move down - Key Down
Move left - Key Left
Move right - Key Right
Shoot - K
